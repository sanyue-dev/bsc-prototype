# 小程序端规格

技术栈：UniApp，目标平台 H5 + 微信小程序。

---

## 车主端（18 屏）

### 底部 Tab Bar（5 项）

| 位置 | Tab | 说明 |
|---|---|---|
| 1 | 首页 | 普通 tab |
| 2 | 省钱 | 普通 tab |
| 3 | 扫一扫 | 居中悬浮大圆按钮（64×64，主色背景） |
| 4 | 商城 | 普通 tab |
| 5 | 我的 | 普通 tab |

> 扫一扫不显示文字标签，以品牌色大圆按钮突出显示。

### 主流程
扫码 → 站点详情 → 确认充电（选券）→ 充电中 → 完成结算

### 屏幕清单

| 屏幕 | 类型 | 关键内容 |
|---|---|---|
| 首页 | 原生 | 快捷入口（优惠券/报修/客服）、活动 Banner 轮播、搜索栏、附近站点列表、当前充电卡 |
| 扫一扫 | 原生 | 摄像头扫码，支持相册/手动输入桩号 |
| 地图找站点 | 原生 | 地图大头针、筛选器（空闲优先/24h/雨棚等）|
| 站点详情 | H5 | 端口状态矩阵（4列网格）、计费卡片、站点信息 |
| 完整计费规则 | H5 | 档位说明、计费细则、余额校验 |
| 确认充电 | H5 | 计费确认、优惠券开关、余额校验、启动按钮 |
| 充电中 | 原生 | 环形表盘+计时、实时功率（动态显示当前档位）、服务费/电费拆分、功率波形 |
| 充电完成 | 原生 | 结算票根、费用明细、碳减排 |
| 我的订单 | H5 | 订单列表（状态筛选：全部/充电中/已完成/退款/异常）、本月统计 |
| 订单详情 | H5 | 充电过程时间线、完整费用拆分 |
| 我的钱包 | H5 | 余额展示、充值套餐、账户明细 |
| 优惠券/月卡 | H5 | 持有券列表、月卡状态与续费 |
| 省钱中心 | H5 | 每日签到、我的权益（券/月卡）、限时活动、推荐有奖 |
| 积分商城 | H5 | 商品分类、限时秒杀（倒计时）、积分兑换；Tab Bar 高亮「商城」 |
| 消息通知 | H5 | 充电完成、余额告警、系统通告 |
| 个人中心 | H5 | 用户信息、充电统计、服务入口（钱包/券/消息/订单快捷卡） |
| 设备报修 | H5 | 故障类型、描述、照片上传 |
| 客服中心 | H5 | AI 客服、电话、常见问题 |

### 端口状态显示
- 🟢 **空闲** — 可点击，进入确认充电
- 🔵 **使用中** — 不可点击，显示脉冲动画
- 🔴 **故障** — 不可点击，提示报修

### 充电启动条件
- 账户余额 ≥ **¥3.00**（最低预存额，可由计费规则配置）
- 月卡用户免余额限制
- 不满足时弹出充值引导，不允许直接启动

### 订单状态（UI 展示标签）
| 展示标签 | 含义 |
|---|---|
| 充电中 | 充电进行中（对应后端订单状态「待结算」） |
| 已完成 | 充电结束并结算 |
| 退款 | 已退款 |
| 异常 | 异常终止 |

> 「充电中」为前端 UX 简化标签，后端订单实体对应状态为「待结算」，见 `docs/data-model.md`。

## 运营商端（5 屏）

通过 Tweaks 切换 `role: "operator"` 进入。

### 底部 Tab Bar（4 项）

| Tab | 屏幕 |
|---|---|
| 看板 | 运营看板 |
| 站点 | 站点管理 |
| 设备 | 设备监控 |
| 工单 | 故障工单 |

> **计费配置**不在 Tab Bar 中，通过 Tweaks 面板「跳转至」导航访问。

### 屏幕清单

| 屏幕 | 关键内容 |
|---|---|
| 运营看板 | 今日营收 sparkline、KPI 四格（订单/用电/设备在线率/故障工单）、站点 TOP5、实时告警 |
| 站点管理 | 名下站点列表、设备数/利用率/今日收益、状态筛选 |
| 设备监控 | 设备热力格子图（充电中/空闲/离线/故障）、总负载折线图、异常设备+一键派单 |
| 故障工单 | 待处理/处理中/已完成分 Tab、SLA 倒计时、接单/转派 |
| 计费配置 | 计费周期选择（预设 3/5/6/10 分钟 + 自定义）、各档位每周期费用、计费规则预览 |
