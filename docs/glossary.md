# 术语规范

## 核心术语

| 术语 | 字段名 | 定义 |
|---|---|---|
| 设备 | `device` | 一台主机箱，4～12 个端口，整机维持心跳连接 |
| 端口 | `port` | 单个充电插座，状态：空闲/使用中/故障 |
| 站点 | `station` | 一个物理地点，包含若干设备 |
| 充电会话 | `charging_session` | 物理充电过程（高频写入），不含支付 |
| 充电订单 | `charging_order` | 商业结算记录（低频写入），含费用拆分 |

## 指标定义

| 指标 | 公式 | 状态 |
|---|---|---|
| 设备在线率 | 心跳正常设备数 ÷ 总设备数 | ✅ 正确 |
| 端口利用率 | 使用中端口数 ÷ 总端口数 | ✅ 正确 |
| 端口故障率 | 故障端口数 ÷ 总端口数 | ✅ 正确 |
| 端口在线率 | — | ❌ **不存在**，端口无独立网络连接 |

## ⛔ 严禁出现的词汇和图示

- 汽车、轿车、新能源车
- 充电枪（汽车含义）
- 充电桩（汽车语境）
- 充电柜门、储物柜
- 端口在线率

> 本产品充电设备无柜门，用户仅将充电器插入公共插座，类似共享插线板。
